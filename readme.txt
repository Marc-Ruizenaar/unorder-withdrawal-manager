EU-compliant statutory withdrawal handling for WooCommerce, aligned with Dutch Civil Code Article 230oa (cooling-off and right of withdrawal).

== Description ==

Un Order helps WooCommerce stores offer a clear withdrawal flow for eligible orders: customers can request withdrawal from their account, and shop staff can review and approve or reject requests with email notifications.

**Free version**

* WooCommerce settings tab for EU withdrawal options (including category exclusions where applicable).
* My Account integration for customers to submit withdrawal requests.
* Admin list to manage requests and related WooCommerce emails (received, approved, rejected, admin notices).

**Un Order Pro** (optional add-on) adds guest order lookup, custom withdrawal periods, and licensing.

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/` or install the ZIP from the WordPress Plugins screen.
2. Activate the plugin.
3. Ensure WooCommerce is installed and active.
4. Go to WooCommerce → Settings → Un Order to configure withdrawal options.

== Frequently Asked Questions ==

= Does this work without WooCommerce? =

No. Un Order requires WooCommerce.

= Where do I configure withdrawal rules? =

In the WooCommerce admin under WooCommerce → Settings, on the Un Order tab.
